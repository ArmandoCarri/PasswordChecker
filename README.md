# PasswordChecker
Description: Program will prompt the user to enter a password. After checking the given password for letters, numbers, special characters, and size, the program will determine how strong the password is based on a scoring system.
How to run: The program was created in C# using Jetbrain Rider IDE. For easiest use, it would be best to open the solution (.sln) file using the Rider IDE. This would allow seamless program use and terminal prompting.
