# PasswordChecker

Description: Program will prompt the user to enter a password. After checking the given password for letters, numbers, special characters, and size, the program will determine how strong the password is based on a scoring system.

How to run: The program was created in the Jetbrain Rider IDE using .NET SDK 9.0. The optimal way to run the program is through the Rider IDE, opening the 'PasswordChecker.sln' with the .NET SDK 9.0 installed. The program can be ran in the terminal with .NET SDK 9.0 installed. To run in the terminal: download the folder, navigate to the 'PasswordChecker' folder containing the 'PasswordChecker.csproj' file, and run the program using the terminal command 'dotnet run'.
